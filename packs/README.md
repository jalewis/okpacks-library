# Packs

Each subdirectory is an OKEngine pack **definition** (an empty vault). Deploy one with
OKEngine — see the root [README](../README.md) and the OKEngine
[authoring guide](https://github.com/jalewis/okengine/blob/main/docs/authoring-a-pack.md).

| Pack | Domain | Status |
|---|---|---|
| [`okpack-example`](okpack-example) | generic starter — the skeleton shape, ready to fill | example |
| [`okpack-competitive`](okpack-competitive) | competitive / market intelligence for **any** tracked market — competitors, products, segments, deals, signals; method-only (identity at deploy time) | **flagship (generic)** |
| [`okpack-ai-research`](okpack-ai-research) | AI / LLM research watch — models, labs, researchers, techniques, benchmarks, predictions | community |
| [`okpack-cti`](okpack-cti) | security / threat-intel, STIX-aligned — the OKF-CTI standard (spec + conformance + STIX/OCSF projectors) | flagship |
